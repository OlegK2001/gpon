# Как внести вклад

Спасибо за интерес к проекту GPON Network Simulator! Мы приветствуем вклад от всех желающих.

## Как внести свой вклад

### Сообщение об ошибках

Если вы нашли ошибку:

1. Проверьте, не была ли она уже зарегистрирована
2. Создайте новый Issue с описанием
3. Приложите логи и скриншоты (если применимо)

### Предложение функций

Идеи новых функций приветствуются:

1. Создайте Feature Request на GitHub
2. Опишите предполагаемое поведение
3. Объясните, почему это полезно

### Pull Requests

Перед отправкой PR:

1. Создайте ветку от `main`
2. Внесите изменения
3. Напишите тесты (если применимо)
4. Обновите документацию
5. Убедитесь, что код работает

### Стиль кода

**Python**:
- PEP 8
- Docstrings для всех функций
- Type hints где возможно

**TypeScript**:
- ESLint rules
- Явные типы
- Комментарии для сложной логики
